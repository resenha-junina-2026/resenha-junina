🔥 Configurar Firebase

Com o Firebase, qualquer celular acessa a lista em tempo real — sem precisar do mesmo dispositivo.

📋 Passo a passo:
  1. Acesse console.firebase.google.com
  2. Clique "Criar projeto" → dê um nome → desative o Google Analytics
  3. No menu lateral, clique "Firestore Database" → Criar banco → Modo de teste
  4. Clique no ícone </> (Web) → registre o app → copie o objeto firebaseConfig
  5. Abra o arquivo ResenhaJunina2026.html num editor de texto
  6. Procure FIREBASE_CONFIG e substitua pelos valores copiados
  7. Salve e suba o arquivo atualizado no GitHub
💡 O plano gratuito do Firebase suporta até 50.000 leituras/dia — mais que suficiente para a festa.
